# Changelog

## v0.1.0

首个可用版本，面向微信公众号文章的手帐涂鸦风配图工作流。

### 新增

- 支持把中文文章选题、草稿或完整文章转成公众号配图。
- 默认直接生成成品图，同时支持按需输出 Prompt。
- 支持封面图和内文图两类场景：
  - 封面图默认比例：`2.35:1`
  - 内文图默认比例：`16:9`
- 新增 OpenAI Codex 移动端文章示例：
  - 1 张封面图
  - 3 张内文图
- 新增中英文 README 和语言跳转入口。
- 新增 `references/style-guide.md`，沉淀笔记本纸、粗黑手写字、手绘 UI 卡片和少量红色点缀等风格规则。

### 优化

- 默认整套配图流程改为先生成 1 张封面图确认风格，再继续生成内文图。
- 明确 `assets/examples/` 为当前风格基准，后续生成应对齐示例图的纸张质感、手写标题和线条风格。
- 强化 Prompt 质量规则：即使直接出图，也要先内部生成完整高质量 Prompt。
- 加强负面约束，避免生成干净数字字体、SVG 风格、企业插画、3D 渲染或复杂信息图。

### 使用建议

普通公众号文章推荐这样调用：

```text
用 $inknote-wechat 给这篇文章出一套公众号配图：1 张封面 + 3 张内文图。
```

如果只想要提示词：

```text
用 $inknote-wechat 给这篇文章输出配图提示词，不要出图。
```

### Release Notes

`v0.1.0` 确立了 inknote-wechat 的核心产品形态：它不是单纯的提示词模板，而是一套面向公众号内容生产的配图技能。这个版本重点解决三件事：稳定的手帐涂鸦风格、适合微信阅读的图片比例，以及更接近真实设计流程的“先看封面，再扩展整组”工作方式。
