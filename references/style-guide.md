# Notebook Doodle Style Guide

## Visual DNA

- Base surface: white notebook paper, pale blue ruled horizontal lines, one red vertical margin line, subtle scan or paper shadow.
- Linework: thick black marker or felt-tip pen, uneven pressure, wobbly edges, hand-drawn boxes, arrows, icons, and small characters.
- Composition: editorial sketch page, not a polished UI mockup. Use generous blank space and one clear focal object.
- Text: hand-lettered or chunky marker lettering. Keep visible words short and intentional.
- Color: mostly black and white. Add only one or two small accents such as red flag, yellow highlight, blue sticker, or green check.
- Details: tiny marginal notes, arrows, mini portraits, cursor pointer, little UI cards, coffee cup, folders, robot, checklist, sticky note, browser window.

## Good Subject Translations

- AI workflow: messy desk, prompt note, small robot assistant, arrows from idea to output.
- Vibe Coding: notebook sketch of an app window, cursor, code block as doodle lines, builder character pushing blocks.
- Product thinking: hand-drawn funnel, checklist, user face bubbles, before/after boxes.
- Design process: wireframe cards, sticky notes, tiny annotation arrows, magnifier, swatches.
- Tool tutorial: step-by-step boxes stacked like notebook UI buttons.

## Must Avoid

- Polished vector illustration, flat SaaS landing-page art, 3D clay render, photorealistic desk photo.
- Dense diagrams that cannot be read on a phone.
- Perfect geometric UI, glossy gradients, neon cyberpunk, generic AI brain, overused glowing network nodes.
- Too many colors. The style should read as black pen on lined paper first.
- Childish school worksheet style. Keep it handmade but sharp, like a designer's rough concept board.

## Prompt Add-ons

Use these clauses when needed:

- For covers: `bold hand-lettered Chinese headline, centered focal sketch, enough empty paper around the subject for WeChat crop safety`.
- For section images: `small editorial spot illustration, no large headline, one compact metaphor`.
- For tool/process images: `stacked hand-drawn UI cards, rough checkbox marks, arrows showing flow`.
- For contrast images: `split notebook page into left and right imperfect boxes, before and after labels`.

## Negative Prompt

```text
polished vector art, corporate illustration, 3d render, photorealistic, glossy gradient, neon glow, perfect geometry, clean digital UI mockup, stock image, cute mascot, anime, watercolor, oil painting, dense text, tiny unreadable labels, cluttered infographic
```
